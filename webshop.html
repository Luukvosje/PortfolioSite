WebshopProject

English

For this school assignment I made a simple javascript webshop.

I have created a constructor so that I can create an infinite number of products. If there are no more products, it will be out of stock. You will also see the VAT when you click on checkout.

It was a simple assignment that I made in a week.

You can see the website at the bottom of the page.
